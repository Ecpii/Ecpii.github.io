Reading Files

The cat command will read the contents of a file argument when provided with just one argument.

cat [OPTION]… [FILE]…

The name of the command comes from the word concatenate, which means to join together. When provided with multiple files, it will combine the contents of them and push them to STDOUT.

Replacing Contents

The tr command, or translate command will change one set of characters to another. It is analogous to 'find and replace'.

tr [OPTION]… SET1 [SET2]

As with other commands, you can specify a range of characters for a set:

cat alpha-first.txt | tr 'a-z' 'A-Z'

A IS FOR ANIMAL
B IS FOR BEAR
C IS FOR CAT
D IS FOR DOG
E IS FOR ELEPHANT
F IS FOR FLOWER
